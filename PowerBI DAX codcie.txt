Misure e Colonne Calcolate per Agenzia
1. Numero di annunci pubblicati da ciascuna agenzia
NumeroAnnunciPubblicati = 
CALCULATE(
    COUNTROWS(annuncio),
    RELATEDTABLE(agente)
)

2. Numero di affitti gestiti da ciascuna agenzia

NumeroAffittiGestiti = 
CALCULATE(
    COUNTROWS(affitti),
    RELATEDTABLE(agente)
)

3. Prezzo medio degli affitti gestiti da ciascuna agenzia

PrezzoMedioAffitti = 
CALCULATE(
    AVERAGE(affitti[prezzo]),
    RELATEDTABLE(agente)
)

4. Superficie media degli affitti gestiti da ciascuna agenzia

SuperficieMediaAffitti = 
CALCULATE(
    AVERAGE(affitti[superficie]),
    RELATEDTABLE(agente)
)
5. Numero di preferenze per gli annunci di ciascuna agenzia
NumeroPreferenze = 
CALCULATE(
    COUNTROWS(preferenze),
    RELATEDTABLE(annuncio),
    RELATEDTABLE(agente)
)

6. Numero di utenti associati a ciascuna agenzia

NumeroUtentiAssociati = 
CALCULATE(
    COUNTROWS(agente),
    RELATEDTABLE(utente)
)

7. Descrizione delle tipologie di affitti gestiti da ciascuna agenzia

TipologieAffittiGestiti = 
CONCATENATEX(
    VALUES(affitti[idtipologia]),
    RELATED(tipologia[descrizione]),
    ", "
)
